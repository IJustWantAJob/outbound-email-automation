Nightly Automation

The nightly agent uses your Anthropic API key to autonomously research contacts, write emails, and commit to git.

Option 1: Run manually (easiest)

bash nightly/find_more_contacts.sh --list   # see your segments
bash nightly/find_more_contacts.sh segment_name 10  # find 10 contacts

Option 2: Cloud server with cron (autonomous)

Deploy to a $5/mo server (DigitalOcean, EC2, Hetzner). Runs at 3am daily.

bash nightly/deploy.sh  # one-command setup

Option 3: GitHub Actions (free)

Add your Anthropic key as a repo secret and use a scheduled workflow. No server needed.

Note: Cron jobs require an always-on machine. A closed laptop won't run scheduled tasks.
